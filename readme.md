<h2><b></b>Malii TaxAdvisor<b></h2> <br>

<b>อธิบายผลงาน:</b> <br>
-Malii เป็น Line Chatbot ที่พัฒนาขึ้นเพื่อเป็นผู้ช่วยในการให้คำปรึกษาด้านภาษีและเครื่องมือช่วยคำนวณภาษีเงินได้บุคคลธรรมดา โดยออกแบบมาให้ใช้งานได้ง่ายผ่านแพลตฟอร์ม Line 
ซึ่งคนไทยใช้มากกว่า 78.2% ของประชากร ช่วยแก้ไขปัญหาความซับซ้อนของภาษี ความเข้าใจผิด และการขาดข้อมูลเกี่ยวกับการยื่นภาษีออนไลน์<br><br>

<b>จุดเด่นของผลงาน:</b><br>
-ใช้งานผ่าน Line ซึ่งเข้าถึงง่ายและคุ้นเคยสำหรับคนไทย<br>
-ให้คำปรึกษาได้หลากหลายทั้งคำถามทั่วไปและเทคนิคเฉพาะ<br>
-มีลิงก์เข้าสู่ระบบคำนวณภาษีและยื่นภาษีออนไลน์<br>
-เชื่อมต่อกับ Google Gemini API ช่วยให้ตอบคำถามได้ฉลาดขึ้น<br>
-ใช้ Google Sheet และ Google App Script ในการควบคุมการตอบกลับ<br><br>
  
<b>ฟีเจอร์ ฟังก์ชัน การทำงาน:</b><br>
-ฟังก์ชันถามตอบเรื่องภาษีเงินได้บุคคลธรรมดา<br>
-ลิงก์สำหรับเข้าสู่หน้าเว็บคำนวณภาษี ( https://pacharapong.kimson.us/tax_calculator/ )<br>
-ลิงก์เข้าสู่ระบบยื่นภาษีออนไลน์ (  https://efiling.rd.go.th/rd-efiling-web/login )<br><br>

<b>คุณภาพ และมาตรฐานของผลงาน:</b><br>
  -มีความแม่นยำจากการใช้ API ปัญญาประดิษฐ์ (Google Gemini)<br>
  -ได้รับแรงบันดาลใจจาก "น้องอารี" ของกรมสรรพากร และพัฒนาต่อยอดเพื่อให้มีความยืดหยุ่นมากขึ้น<br>
  -ใช้เครื่องมือจริงในการพัฒนา เช่น LINE Developer, Google App Script, WordPress<br>
